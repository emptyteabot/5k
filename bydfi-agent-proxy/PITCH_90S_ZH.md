# 90 秒路演稿

我要解决的问题很直接：
BYDFI 这种复杂业务里，不管是员工还是用户，很多时候都不是不会点按钮，而是不知道该怎么理解信息、该找谁、该怎么把一句话变成一个明确动作。

所以我做的不是一个普通聊天机器人。
我做的是一个 **BYDFI 定制版 Hermes Agent 网页终端**。

表面上，它就是一个极简中文输入框。
但底层接的是 Hermes 的记忆、skills、工具调用和执行能力，再叠加我已经整理进来的 BYDFI 资料、报告流和工作脚本。

这意味着它不是只会总结，而是能把一句自然语言往下压，变成结构化理解、结果判断和下一步动作。

这次黑客松我故意不讲大而空的全能平台，我只抓一个最容易让评委看懂的方向：
让一个复杂的任务，从“人自己理解和拼装”，变成“先让 Agent 跑第一轮”。

现场大家看到的不是后端代码，而是一个很直接的结果：
输入一句话，系统立刻拆意图、过边界、出结果。

这套东西的价值，不在于它多会聊天，
而在于它把复杂能力压成了一个可以直接用、可以部署、可以继续扩展的 BYDFI AI 终端。
